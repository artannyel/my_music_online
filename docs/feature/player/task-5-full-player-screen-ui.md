# Task 5: Presentation - Tela FullPlayerView Estilo YouTube Music (Stitch Design)

## 📌 Descrição Aprofundada
Desenvolver a tela completa de reprodução do player (*Full Player View*) inspirada na interface do **YouTube Music** e prototipada no [Stitch](https://stitch.withgoogle.com/projects/16430281818792447633).

## 🎯 Escopo da Task
1. Criar `lib/src/features/player/presentation/views/full_player_screen.dart`:
   - Botão para recolher (*Dismiss / Slide down*).
   - Capa do Álbum grande em alta resolução com sombra neon ambiente no fundo.
   - Nome da música e artista com links interativos para ir à página do Álbum e Artista.
   - Scrubber / Slider de tempo com tempo decorrido e restante.
   - Botões principais: Shuffle, Anterior, Play/Pause principal estilizado, Próxima, Repeat.
   - Menu contextual (três pontos): "Adicionar à Playlist", "Ir para Álbum", "Ir para Artista", "Equalizador".
   - Aba deslizante inferior com a Fila de Músicas (*Up Next / Queue*).

## 📋 Arquivos a Modificar / Criar
- `lib/src/features/player/presentation/views/full_player_screen.dart`

## ✅ Critérios de Aceite
