# Task 4: Presentation - Interface de Usuário HomeScreen (Stitch Design)

## 📌 Descrição Aprofundada
Desenvolver a interface gráfica da tela principal (Home) alinhada ao protótipo do [Stitch](https://stitch.withgoogle.com/projects/16430281818792447633).

## 🎯 Escopo da Task
1. Criar `lib/src/features/home/presentation/views/home_screen.dart`:
   - Header com saudação baseada no horário (Bom dia, Boa tarde, Boa noite) e avatar do usuário logado.
   - Carrossel horizontal de "Sugestões para Você" com efeito glassmorphism e sombras neon.
   - Seção vertical "Mais Tocadas" com número de ranking, capa miniatura, título, artista e botão de 3 pontos.
   - Suporte a *Pull-to-refresh*.
   - Ação ao clicar em qualquer item: iniciar reprodução de mídia ou navegar para a tela de Álbum/Artista.

## 📋 Arquivos a Modificar / Criar
- `lib/src/features/home/presentation/views/home_screen.dart`

## ✅ Critérios de Aceite
- Tela Home fluida, responsiva, com visual escuro elegante e navegação responsiva.
