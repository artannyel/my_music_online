# Task 4: Presentation - Interface de Usuário EqualizerScreen (Stitch Design)

## 📌 Descrição Aprofundada
Desenvolver a interface gráfica da tela de equalização inspirada nos controles do [Stitch](https://stitch.withgoogle.com/projects/16430281818792447633).

## 🎯 Escopo da Task
1. Criar `lib/src/features/equalizer/presentation/views/equalizer_screen.dart`:
   - Header com título "Equalizador" e chave geral de Ligar/Desligar (*Master Switch*).
   - Chips horizontais com os presets (Flat, Bass Boost, Rock, Pop, Jazz, Eletrônica).
   - Sliders verticais interativos de frequência (60Hz, 230Hz, 910Hz, 3.6kHz, 14kHz) com marcadores de decibéis (-12dB a +12dB).
   - Botão para resetar as frequências para o padrão.

## 📋 Arquivos a Modificar / Criar
- `lib/src/features/equalizer/presentation/views/equalizer_screen.dart`

## ✅ Critérios de Aceite
- Interface estilosa em tema escuro, sliders fluidos e responsivos.
